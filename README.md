# Royal Court Sports Competition 2026

A modern React application for member registration for the 2026 Sports Competition.

## Tech Stack
- Frontend: React + Vite + TypeScript + TailwindCSS
- Database: Supabase
- Hosting: Vercel

## Running Locally
1. `npm install`
2. Create `.env.local` and add your keys:
   - `VITE_SUPABASE_URL=...`
   - `VITE_SUPABASE_ANON_KEY=...`
3. `npm run dev`
